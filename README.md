## 📁 How to download the project 
´``bash 
    #clone the repository
     $ git clone https://github.com/VidalJunior05/to-do-list-javascrip.git

    # Enter directory
    $ cd to-do-list

    # install dependencies
    $ yarn install

    # Initialize project
    $ yarn start

```
